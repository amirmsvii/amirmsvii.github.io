Featured Projects

iCleanLines Board Program Firmware

Production-Ready IoT Water Management System

Engineered a sophisticated embedded firmware solution for enterprise-grade water management systems. Built with advanced multi-tasking architecture using FreeRTOS, featuring real-time communication, offline resilience, and autonomous operation capabilities.

Key Technical Achievements

Real-time Communication

WebSocket-based bidirectional communication with backend systems, featuring automatic reconnection and exponential backoff

Multi-tasking Architecture

Three specialized FreeRTOS tasks running concurrently: DataTask (Priority 3), FlushTask (Priority 5), WebSocketTask (Priority 4)

Offline Data Caching

Smart batch storage to SD card with automatic sync when connectivity is restored, ensuring zero data loss

Advanced Hardware Integration

RFID readers, UART communication, environmental sensors, LED controllers, and real-time monitoring systems

System Architecture

DataTask

UART data processing, RFID integration, system health monitoring, and periodic sensor readings

FlushTask

Automated flush cycle execution, schedule management, and precise duration control

WebSocketTask

Real-time command processing, connection management, and status broadcasting

Technical Specifications

Platform: ESP32 with PlatformIO framework
RTOS: FreeRTOS with concurrent task execution
Communication: WiFi, WebSocket, UART, I²C protocols
Storage: SD Card + LittleFS hybrid storage system
Features: OTA updates, web configuration portal, real-time diagnostics
